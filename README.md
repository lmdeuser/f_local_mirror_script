# Fedora Local Mirror Scripts

> [!WARNING]
> **ВНИМАНИЕ:** Данные скрипты в настоящий момент находятся в **режиме тестирования (BETA)**.
> Используйте их на свой страх и риск. Рекомендуется проверять логи (`/var/log/fedora-mirror-setup.log` и `/var/log/fedora-mirror-update.log`) при первых запусках.

Этот репозиторий содержит набор bash-скриптов для создания и поддержания актуального локального зеркала репозиториев Fedora и RPM Fusion. Скрипты автоматически загружают только новейшие пакеты (`--newest-only`), экономя место на диске.

## Состав
- `setup-local-mirror.sh` — скрипт первичной настройки. Устанавливает необходимые зависимости, подготавливает структуру директорий, делает первичную синхронизацию (через быструю параллельную загрузку DNF) и прописывает конфигурацию локального зеркала (`/etc/yum.repos.d/local-mirror.repo`).
- `update-local-mirror.sh` — скрипт для регулярного (например, по cron) обновления зеркала. Он интеллектуально понижает приоритет локального репозитория на время работы (чтобы `dnf` гарантированно скачивал свежие пакеты с официальных серверов, избегая конфликтов), а по завершению — восстанавливает приоритет.

## Требования
- ОС: Fedora Linux
- Запуск с правами суперпользователя (`sudo`)
- Свободное место на диске (по умолчанию скрипт требует минимум 150 ГБ для x86_64, в зависимости от выбранных архитектур).

## Использование
1. Сделайте скрипты исполняемыемыми:
   ```bash
   chmod +x setup-local-mirror.sh update-local-mirror.sh
   ```
2. Запустите первоначальную настройку:
   ```bash
   sudo ./setup-local-mirror.sh
   ```
3. Для последующих обновлений запускайте:
   ```bash
   sudo ./update-local-mirror.sh
   ```
