# Fedora Local Mirror Scripts

> [!TIP]
> **НОВОЕ В ВЕРСИИ 1.3:** Автоматическое определение версии Fedora, поддержка `zstd` и улучшенная логика синхронизации через `dnf reposync`.

Этот репозиторий содержит набор bash-скриптов для создания и поддержания актуального локального зеркала репозиториев Fedora и RPM Fusion. Скрипты автоматически загружают только новейшие пакеты (`--newest-only`), экономя место на диске.

## Основные особенности
- **Динамическое определение ОС**: Скрипты теперь автоматически определяют версию Fedora (через `/etc/os-release`), что делает их переносимыми.
- **Безопасная синхронизация**: При синхронизации локальные репозитории временно отключаются (`--disablerepo='local-*'`), чтобы dnf не пытался «скачивать сам с себя».
- **Оптимизация метаданных**: Используется параллельная работа `createrepo_c` с поддержкой `zchunk` и `zstd` для быстрой генерации кэша.
- **Управление приоритетами**: Локальные зеркала получают высокий приоритет (`priority=5`), а при обновлении приоритеты временно понижаются для гарантированного получения свежих обновлений с внешних зеркал.

## Состав
- `setup-local-mirror.sh` — первичная настройка. Устанавливает зависимости (`createrepo_c`, `dnf-utils`), создает структуру директорий, выполняет первичную синхронизацию и прописывает конфиг `/etc/yum.repos.d/local-mirror.repo`.
- `update-local-mirror.sh` — интеллектуальное обновление. Умеет безопасно понижать приоритет локальных зеркал перед запуском, чтобы избежать конфликтов метаданных.

## Требования
- ОС: Fedora Linux
- Права суперпользователя (`sudo`)
- Место на диске: минимум 150 ГБ для x86_64; для i686 требуется до 200 ГБ.

## Использование
1. Сделайте скрипты исполняемыми:
   ```bash
   chmod +x setup-local-mirror.sh update-local-mirror.sh
   ```
2. Запустите первоначальную настройку:
   ```bash
   sudo ./setup-local-mirror.sh
   ```
3. Для последующих обновлений запускайте:
   ```bash
   sudo ./update-local-mirror.sh
   ```

## Логи
Вы всегда можете проверить ход выполнения в системных логах:
- `/var/log/fedora-mirror-setup.log`
- `/var/log/fedora-mirror-update.log`
